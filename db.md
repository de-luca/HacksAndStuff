# DB

## Polymorphism

```
   +------------------------+      +----------------------+
   | stuff                  |      | stuff_one            |
   +------------------------+      +----------------------+
   | id                     <---+  | id                   |
   | [generic stuff fields] |   +--+ #stuff_id            |
+--+ #stuff_type_id         |   |  | [stuff_one fields]   |
|  +------------------------+   |  +----------------------+
|                               |
|  +------------+               |  +----------------------+
|  | stuff_type |               |  | stuff_two            |
|  +------------+               |  +----------------------+
+--> id         |               |  | id                   |
   | name       |               +--+ #stuff_id            |
   +------------+                  | [stuff_two fields]   |
                                   +----------------------+
```
